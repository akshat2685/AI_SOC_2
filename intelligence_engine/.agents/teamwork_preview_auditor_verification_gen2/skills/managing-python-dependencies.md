## Key takeaways:
1. Always detect package manager (uv, poetry, pipenv, conda, venv+pip).
2. For venv+pip, use explicit path like `.venv/Scripts/pip` or `.venv/Scripts/python` on Windows.
3. NEVER run `pip install` globally.
